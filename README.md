# 🧠 NeuroCards

A lightweight flashcard application focused on **active recall** and **real learning**, not memorizing patterns.

Originally designed to support my girlfriend's study sessions with a more effective approach to retention.

---

## 🚀 Features

- Flip card interaction (question ↔ answer)
- Smooth navigation with slide transitions
- Shuffle mode to prevent memorization by order
- Keyboard support (← → Space / Enter)
- Clean and distraction-free interface

---

## 🧠 Why this project?

Most flashcard tools fail because users start memorizing the **order of answers**, not the content itself.

This project solves that by:
- enforcing interaction (flip)
- allowing randomization of card order
- keeping the UI minimal and focused

---

## 🛠️ Tech Stack

- HTML
- CSS (3D transforms + animations)
- Vanilla JavaScript

No frameworks — fully controlled behavior and performance.

---

## 🎯 Purpose

This project was built to:
- improve real study efficiency
- avoid passive memorization
- create a simple but effective learning tool

---

## 📦 How to use

1. Open `index.html`
2. Load your `cards.json`
3. Start studying

Optional:
- Click **Shuffle** to randomize order
- Use keyboard for faster navigation
